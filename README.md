# بن غيث ERP — نظام إدارة العود والبخور

نظام ERP متكامل لتجارة العود والبخور، ملف HTML واحد، يعمل كـ PWA على iPhone.

## ✨ المميزات

- 🎨 **هوية بن غيث** — ألوان ذهبية كريمية، خط عربي أنيق، ثيم فاخر
- 📊 **Dashboard احترافي** — رأس المال، الكاش، قيمة المخزون، الأرباح، التنبيهات
- 🏬 **4 مخازن** — رئيسي / معرض / احتياطي / شحن + تحويل بينها
- 📦 **منتجات كاملة** — اسم، نوع، دولة، تصنيف، وزن، سعر/كيلو، سعر/جرام، مورد، صور (تلقائياً مضغوطة)، ملاحظات
- 👥 **موردين** — اسم، دولة، هاتف، واتساب، إيميل، تقييم نجوم، ملاحظات، سجل مشتريات تلقائي
- ⚙️ **تصنيفات ديناميكية** — درجات (A/AA/AAA/VIP/Royal/Kinam)، دول، أنواع عود، أنواع منتجات، أنواع مصاريف، مخازن
- 💰 **محاسبة تلقائية**
  - شراء → خصم من الكاش + إضافة للمخزون + قيد في Ledger
  - بيع → خصم من المخزون + إضافة للكاش + حساب الربح
  - مصاريف → خصم من الكاش
  - تحويلات → تحديث مخزن المنتج
- 📒 **Ledger كامل** — كل العمليات بتاريخها ومبلغها وملاحظاتها
- 🚨 **تنبيهات تلقائية** — مخزون منخفض، رصيد سالب
- 📝 **نظام Logs** — Error / Activity / System (آخر 500)
- 🔄 **Patch Updates** — رقم النسخة، تاريخها، التغييرات
- 🌍 **عربي + إنجليزي** — تبديل من الزر العلوي
- 🌙 **Dark Mode** — تبديل من الزر العلوي
- 📱 **PWA** — تثبيت على iPhone Home Screen
- 💾 **تصدير/استيراد** نسخ احتياطية JSON
- 💎 **العملة الافتراضية: د.إ (AED)**

## 📲 رفع المشروع على GitHub Pages

### الطريقة الأولى — من المتصفح (الأسهل)

1. ادخل على المستودع: https://github.com/Saadi80/binGhaith-Oud
1. اضغط **“Add file”** ← **“Upload files”**
1. ارفع ملف `index.html`
1. اعمل **Commit changes**
1. روح **Settings** ← **Pages**
1. تحت **“Source”** اختر:
- Branch: `main`
- Folder: `/ (root)`
1. اضغط **Save**
1. بعد دقيقة، الموقع جاهز على:
   
   ```
   https://saadi80.github.io/binGhaith-Oud/
   ```

### الطريقة الثانية — من Terminal

```bash
git clone https://github.com/Saadi80/binGhaith-Oud.git
cd binGhaith-Oud
# انسخ index.html هنا
git add index.html
git commit -m "Initial release v1.0.0"
git push
```

ثم فعّل GitHub Pages من Settings كما في الخطوة 5–7 أعلاه.

## 📲 تثبيته كتطبيق على iPhone

1. افتح الرابط في **Safari** (مهم: ليس Chrome)
1. اضغط زر المشاركة (المربع مع السهم لأعلى)
1. اختر **“إضافة إلى الشاشة الرئيسية” / “Add to Home Screen”**
1. التطبيق سيظهر بدون شريط Safari ويشتغل offline

## 🗄️ تخزين البيانات

كل البيانات تُحفظ محلياً على جهازك في `localStorage`.

- **مزايا**: خاصة 100%، تشتغل offline، سريعة
- **مهم**: اعمل تصدير دوري للنسخة الاحتياطية (الإعدادات ← تصدير البيانات)

## 🚀 البدء السريع

أول ما تفتح التطبيق:

1. سيطلب منك **رأس المال الابتدائي** بالدرهم
1. ضع رقم رأس مالك النقدي الحالي
1. ابدأ بإضافة موردين ومنتجات

كل عملية شراء منتج جديد ستخصم تلقائياً من رأس مالك وتضيف للمخزون.
كل بيع سيخصم من المخزون ويضيف للكاش ويحسب الربح.
